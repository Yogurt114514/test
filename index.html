Job Descriptions

Simulated Hardware Engineer

  1. Familiar with simulation software TINA and LTspice, able to use them to debug known circuits;
  2. Familiar with analog circuit design, such as low-frequency small signal low noise amplification circuits, high-precision data acquisition, etc;
  3. Strong team collaboration and sharing spirit.

PCB Engineer

  1. Proficient in processor applications such as FPGA, MCU, DSP, etc., including power supply, bus interface, signal detection and amplification design;
  2. Lead the product's single board schematic, PCB design and improvement, as well as BOM file organization according to requirements;
  3. Good coordination and communication skills and team spirit.

Software Engineer

  1. Understand WEB page design and programming such as HTML, CSS, Javascript, etc;
  2. Understand the principles and development of databases, master at least one commonly used database such as SQLServer, MySql, etc., and be able to write SQL and other database statements;
  3. Be able to skillfully use Visual Studio, Eclipse and other development tools for code design;
  4. Good psychological quality and teamwork spirit, rigorous and responsible work attitude;

Vision Algorithm Engineer

  1. Master's degree or above, with in-depth research and unique insights into deep learning algorithms, computer vision and other fields;
  2. Strong programming skills, familiar with image development related platforms such as C#/C++.Python.Halcon.Opencv;
  3. At least command one framework of TensorFlow.Caffe, Torch，FastRCNN.YOLO etc;
  4. Good command of IDE, such as Anconda/pycharm etc，those with GPU.Cuda.Cudnn programing experience are preferred.
